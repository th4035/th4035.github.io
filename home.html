플러그인 소개
GSWeb 플러그인
GSWeb 플러그인은 Rhino 환경에서 반복 작업을 줄이고 설계 업무의 효율을 높이는 통합 지원 도구입니다.
시작 안내
GSWeb 플러그인 빠른 시작
플러그인을 설치하면 Rhino에 GSWeb 툴바가 자동으로 생성되며, 해당 툴바의 버튼을 통해 즉시 기능을 실행할 수 있습니다.
1

플러그인 설치

YAK 파일을 Rhino에 드래그하여 플러그인을 설치합니다.

2

계정 연결

등록된 이름과 암호를 입력하여 인증 절차를 진행합니다.

3

툴바 확인

설치 완료 후 Rhino에 생성된 GSWeb 플러그인 툴바를 확인합니다.

4

기능 실행

툴바 아이콘을 클릭하여 원하는 기능을 실행합니다.

추가 안내
재설치 및 업데이트
최신 기능 적용 또는 오류 해결을 위해 기존 플러그인을 제거한 후 최신 YAK 파일을 다시 설치합니다.
1

기존 플러그인 삭제

Rhino에서 'PackageManager'를 실행한 후 GSWeb을 선택하여 제거합니다.

2

UI 정리

Rhino 툴바 정리 도구(.exe)를 실행하여 이전 버전 UI 잔여 파일을 정리합니다.

3

재설치

기존 플러그인 다운로드 방법과 동일하게 설치합니다.

문제 해결
설치 오류 안내
설치 또는 업데이트 과정에서 오류가 발생하는 경우 아래 내용을 확인하십시오.
1

Rhino 버전 확인

Rhino 7 SR38 이상에서만 설치 가능합니다. 버전이 낮은 경우 업데이트 후 다시 시도하십시오.

2

인증 오류

인증 과정에서 오류가 발생하는 경우, 기술전략팀에 문의하여 주시기 바랍니다.

3

사용 환경 제한

본 플러그인은 회사 지정 IP 환경에서만 사용 가능하며, 외부 IP에서는 실행이 제한됩니다.
