객체 랜덤 배치

Grasshopper : 선택한 서피스 위에 블록 객체를 랜덤하게 배치하는 기능입니다.

Steps

  • 랜덤 배치할 객체를 블록으로 저장하고 배치할 서피스를 준비합니다.
  • 랜덤 객체 배치 아이콘을 클릭하여 명령을 실행합니다.
  • 객체의 블록 이름을 동일하게 입력합니다.
  • 배치할 서피스를 선택한 후 기준 서피스로 적용합니다.
  • 원하는 옵션에 맞춰 설정한 후 실행 버튼을 클릭합니다.

Note

  • 객체는 블록으로 저장된 경우에만 배치할 수 있습니다.
  • 배치는 선택한 서피스를 기준으로 수행됩니다.
