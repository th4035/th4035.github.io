OpenBOMWindow

OpenBOMWindow : Rhino 모델 내 레이어를 기준으로 부품 객체를 분석하고, 수량 및 규격 정보를 자동 집계하여 BOM을 생성하는 기능입니다.

Steps

  • OpenBOMWindow 명령 실행
    • BOM 입력창이 열립니다.
  • 레이어 불러오기 및 항목 선택
    • 모델 내 레이어를 불러와 BOM에 포함할 부품 항목을 선택합니다.
  • 품명·규격·재질 정보 확인 및 보완
    • 블록명 또는 레이어명을 분석하여 정보를 자동 추출하며,
      누락된 항목은 사용자가 직접 입력해 값을 채워 줍니다.
  • 단가 자동 적용 및 수동 수정
    • “단가 파일 열기” 버튼으로 엑셀 파일을 불러오면,
      부품명과 같은 레이어에 단가가 자동 적용되며, 누락된 항목은 직접 입력할 수 있습니다.
  • 수량·단가 입력 후 동기화 필요
    • 수량이나 단가를 수동으로 입력하거나 객체 수량이 변경된 경우,
      ‘레이어 수량 동기화’ 버튼을 눌러 금액과 수량 정보를 최적화해야 합니다.
  • BOM 내보내기 (Excel)
    • BOM 목록을 Excel (.xlsx) 형식으로 저장하고, 파일을 자동으로 실행하여 즉시 확인할 수 있습니다.

Note

  • 단가 파일(.csv 또는 .xlsx)은 부품명과 레이어명이 정확히 일치해야 자동 적용됩니다.
  • 규격 및 재질 정보는 블록명 또는 레이어명을 기반으로 자동 인식되며, 누락된 경우 수동 입력이 필요합니다.
  • 현재 기능은 초기 버전으로, 향후 사용성과 자동화 기능이 지속적으로 개선될 예정입니다.
