HPL_Estimate

HPL_Estimate : 객체를 자동 인식하여 HPL 부재를 탐색하고 견적을 계산합니다.

HPL_Calculator : 직접 부재 정보를 입력하여 HPL 견적을 수동으로 계산할 수 있습니다.

Steps

  • HPL_Estimate 명령 실행
  • 견적을 낼 HPL 객체 선택
  • 자동 계산된 견적 결과 확인

Note

  • 인식된 두께가 사내 사용 두께(3T, 6T, 10T, 15T)와 일치하지 않을땐 직접 두께값 입력이 필요합니다.
  • 사내 사용 두께단가 (1220×2440 기준) : 3T - 45,000 / 6T - 62,000 / 10T - 89,000 / 15T - 118,000(원/m²)
