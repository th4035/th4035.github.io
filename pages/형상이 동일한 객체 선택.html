SelectAllDuplicates

SelectAllDuplicates : 형상이 같은 객체 자동 선택

SelectSameShape : 겹쳐진 동일 형상 자동 선택 (중복된 Brep/서피스/블록 포함)

Steps

  • 형상이 완전히 같은 객체들을 찾기 위한 기준 객체를 선택
  • SelectAllDuplicates 명령 실행
  • 동일한 형상 수량 및 결과 확인

Note

  • 인식된 두께가 사내 사용 두께(3T, 6T, 10T, 15T)와 일치하지 않을땐 직접 두께값 입력이 필요합니다.
  • 사내 사용 두께단가 (1220×2440 기준) : 3T - 45,000 / 6T - 62,000 / 10T - 89,000 / 15T - 118,000(원/m²)
